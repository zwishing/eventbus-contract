# EventBus 0.2.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship breaking 0.2.0 that addresses every P0+P1 in the review: workspace split, dyn-safe core traits, consume-self Delivery, newtype identifiers, BatchOutcome, limiter race fix, error chain hygiene, tracing/observer hooks, and a complete migration story.

**Architecture:** Migrate the single `eventbus-contract` crate into a 6-crate workspace under `crates/`. Reshape `Publisher` / `Subscriber` / `Handler` to be object-safe via `BoxFuture`, add `*Ext` traits for monomorphized convenience, split `Delivery` into a read-only `Delivery` + consume-self `DeliveryControl` (delivered to handlers as `Box<dyn DeliveryHandle>`). Each PR compiles and `cargo test --workspace` is green before the next PR starts.

**Tech Stack:** Rust 1.82, Tokio (sync/macros/rt/time), `bytes`, `chrono`, `dashmap`, `thiserror`, optional `redis` + `serde_json`, optional `tracing`. Workspace dependency inheritance via `[workspace.dependencies]`.

**Spec:** [`docs/superpowers/specs/2026-05-06-eventbus-0.2.0-design.md`](../specs/2026-05-06-eventbus-0.2.0-design.md)

**PR roadmap:**

| 2 | Object-safe core traits + `*Ext` | Public trait surface uses `BoxFuture`; existing call sites migrated |

```rust
async fn publish_inner(
    &self,
    message: Message,
    options: &PublishOptions,
) -> Result<MessageId, EventBusError> {
    options.validate()?;
    if let Some(delay) = options.delay { tokio::time::sleep(delay).await; }
    let message = Self::prepare_message(message, options, self.options.max_payload_bytes)?;
    let topic = message.topic.clone();
    let id = self.backend.publish(&topic, message).await?;
    Ok(MessageId::new(id))
}
```

Rename existing `publish_batch` body to `publish_batch_impl` and adjust to **temporarily** still return `Err` on first failure (PR 5 will rebuild it to populate `BatchOutcome`):
```rust
async fn publish_batch_impl(
    &self,
    msgs: Vec<Message>,
    opts: PublishOptions,
) -> Result<BatchOutcome, EventBusError> {
    // Temporary shim: preserves 0.1 fail-fast behavior; PR 5 replaces this
    // with a per-message Result vec.
    opts.validate()?;
    if let Some(delay) = opts.delay { tokio::time::sleep(delay).await; }
    let max_payload_bytes = self.options.max_payload_bytes;
    let prepared: Vec<Message> = msgs
        .into_iter()
        .map(|m| Self::prepare_message(m, &opts, max_payload_bytes))
        .collect::<Result<Vec<_>, _>>()?;
    let mut results = Vec::with_capacity(prepared.len());
    for m in prepared {
        let topic = m.topic.clone();
        match self.backend.publish(&topic, m).await {
            Ok(id) => results.push(Ok(MessageId::new(id))),
            Err(e) => results.push(Err(e)),
        }
    }
    Ok(BatchOutcome { results })
}
```

```rust
        self: Box<Self>,
        reason: crate::BoxedError,
    ) -> crate::BoxFuture<'static, Result<(), EventBusError>> {
        Box::pin(async move {
            let r = reason.to_string();
            self.publish_dead_letter(&r).await?;
            self.mark_acked().await
        })
    }

    fn retry(
        self: Box<Self>,
        reason: crate::BoxedError,
    ) -> crate::BoxFuture<'static, Result<(), EventBusError>> {
        Box::pin(async move {
            let retry_exhausted = self.state.attempt >= self.state.max_attempt;
            let reason_str = reason.to_string();
            if retry_exhausted {
                if self.config.dead_letter_topic.is_none() {
                    return Err(EventBusError::Validation(format!(
                        "retry exhausted ({}/{}) but no dead_letter_topic configured for topic {}",
                        self.state.attempt, self.state.max_attempt, self.config.topic
                    )));
                }
                self.publish_dead_letter(&reason_str).await?;
            } else {
                let mut retried = Message::clone(&self.message);
                retried.headers.insert(HEADER_RETRY_ATTEMPT.to_string(), self.state.attempt.to_string());
                retried.headers.insert(HEADER_RETRY_REASON.to_string(), reason_str);
                self.backend.publish(&self.config.topic, retried).await?;
            }
            self.mark_acked().await
        })
    }
}
```

```rust
async fn handle_delivery(
    handler: &dyn Handler,
    delivery: Box<dyn DeliveryHandle>,
    ack_mode: AckMode,
) -> Result<(), EventBusError> {
    // 0.1 had two auto-ack modes; with consume-self semantics we can't both
    // hand the box to the handler AND finalize behind their back. So:
    //   AutoOnHandlerSuccess: caller code below finalizes after handler runs.
    //   Manual: handler finalizes itself; if it doesn't, permit drops on box drop.
    //   AutoOnReceive: see Task 3.3.
    match ack_mode {
        AckMode::Manual | AckMode::AutoOnReceive => {
            // For AutoOnReceive in this PR we treat the ack as fire-and-forget
            // before delivery; handled in Task 3.3.
            handler.handle(delivery).await
        }
        AckMode::AutoOnHandlerSuccess => {
            // Split into a thin wrapper so we can finalize.
            run_with_auto_finalize(handler, delivery).await
        }
    }
}
```

- [ ] **Step 5: Implement `run_with_auto_finalize`**

```rust
async fn run_with_auto_finalize(
    handler: &dyn Handler,
    delivery: Box<dyn DeliveryHandle>,
) -> Result<(), EventBusError> {
    // We need both:
    //   (a) hand the box to the handler so it can read the message
    //   (b) finalize after success (ack) or after error (retry)
    // Since the box can only be moved once, wrap it in an Option-shaped
    // splitter: clone the read-only data needed for fallback, hand the
    // box to the handler — if the handler returns Ok and didn't consume
    // the box, ack; if Err, retry.
    //
    // Pragmatic implementation: introduce an `AutoFinalize` shim that the
    // handler treats as DeliveryHandle. Inside, ack/nack/retry leave a
    // breadcrumb so we know whether to auto-finalize.
    let tracker = AutoFinalizeTracker::new();
    let proxy: Box<dyn DeliveryHandle> = Box::new(AutoFinalizeProxy::new(delivery, tracker.clone()));
    let result = handler.handle(proxy).await;
    if let Some(remaining) = tracker.take_remaining() {
        match &result {
            Ok(()) => remaining.ack().await,
            Err(err) => remaining.retry(Box::new(SimpleError(err.to_string()))).await,
        }?;
    }
    result
}
```

`AutoFinalizeProxy` and `AutoFinalizeTracker` go in `crates/eventbus-core/src/stream/auto_finalize.rs` — they wrap the inner box in a `Mutex<Option<Box<dyn DeliveryHandle>>>` so finalization can happen from either the handler or the consume loop, exactly once.

```rust
async fn publish_batch_impl(
    &self,
    msgs: Vec<Message>,
    opts: PublishOptions,
) -> Result<BatchOutcome, EventBusError> {
    opts.validate()?;
    if let Some(delay) = opts.delay { tokio::time::sleep(delay).await; }

    let max_payload = self.options.max_payload_bytes;
    let prepared: Vec<Result<Message, EventBusError>> = msgs
        .into_iter()
        .map(|m| Self::prepare_message(m, &opts, max_payload))
        .collect();

    let parallelism = prepared.len().clamp(1, self.options.publish_batch_parallelism);
    let mut tasks: tokio::task::JoinSet<(usize, Result<MessageId, EventBusError>)> =
        tokio::task::JoinSet::new();
    let mut iter = prepared.into_iter().enumerate();
    let mut results: Vec<Option<Result<MessageId, EventBusError>>> = Vec::new();

    for _ in 0..parallelism {
        if let Some((idx, prep)) = iter.next() {
            results.push(None);
            let backend = std::sync::Arc::clone(&self.backend);
            tasks.spawn(async move {
                let r = match prep {
                    Err(e) => Err(e),
                    Ok(m) => {
                        let topic = m.topic.clone();
                        backend.publish(topic.as_str(), m).await.map(MessageId::new)
                    }
                };
                (idx, r)
            });
        }
    }

    while let Some(joined) = tasks.join_next().await {
        let (idx, r) = match joined {
            Ok(p) => p,
            Err(je) => {
                // panic in publish task — record at the next slot we can.
                let idx = results.len();
                results.push(None);
                (idx, Err(EventBusError::source("publish task panicked", je)))
            }
        };
        if idx >= results.len() { results.resize_with(idx + 1, || None); }
        results[idx] = Some(r);

        if let Some((next_idx, prep)) = iter.next() {
            results.push(None);
            let backend = std::sync::Arc::clone(&self.backend);
            tasks.spawn(async move {
                let r = match prep {
                    Err(e) => Err(e),
                    Ok(m) => {
                        let topic = m.topic.clone();
                        backend.publish(topic.as_str(), m).await.map(MessageId::new)
                    }
                };
                (next_idx, r)
            });
        }
    }

    Ok(BatchOutcome {
        results: results.into_iter().map(|o| o.unwrap_or_else(|| {
            Err(EventBusError::Internal("publish_batch slot never filled".into()))
        })).collect(),
    })
}
```

```rust
async fn spawn_messages(
    &self,
    tasks: &mut JoinSet<DeliveryTaskResult>,
    entries: Vec<FetchedEntry>,
    permits: &mut Vec<OwnedSemaphorePermit>,
    runtime: &RuntimeState,
) -> Result<(), EventBusError> {
    for entry in entries {
        match entry {
            FetchedEntry::Decoded(claimed) => {
                let permit = permits.pop().ok_or_else(|| EventBusError::Internal(
                    "spawn_messages out of permits — invariant violation".into()
                ))?;
                let bus = self.clone();
                let config = Arc::clone(&runtime.config);
                let handler = Arc::clone(&runtime.handler);
                let ack_tx = runtime.ack_tx.clone();
                tasks.spawn(async move {
                    bus.process_single_message(config, handler, claimed, permit, ack_tx).await
                });
            }
            FetchedEntry::Malformed { id, error } => {
                self.handle_malformed_entry(&runtime.config, id, error).await;
            }
        }
    }
    // Any unused permits in `permits` drop here, returning their slots.
    Ok(())
}
```
